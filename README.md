# Export Cleaned Dataset Task

This project demonstrates how to export a cleaned dataset into a new CSV file using Python and Pandas.

## Operations Performed

- Loaded the cleaned dataset
- Removed duplicate and empty records
- Checked missing values
- Exported the cleaned dataset
- Reloaded the exported CSV for verification

## Output File

`exported_cleaned_students.csv`

## Tools Used

- Python
- Pandas
- Jupyter Notebook
- Git
- GitHub
